Basic Intensity Transformation Functions

 

Photographic Negative

This is the simplest intensity transformation. Given a grey level image in the range [0,1], a grey point (0) is turned into a white point (1) and vice-versa. This is denoted by f ' (x,y) = 1 - f(x,y) assuming f(x,y) is normalised into the range [0,1].

i.e. for a 256 grey level image, f ' (x,y) = 1 - f(x,y)/256

So, for a black and white pixel with value 70, the new value for the pixel will be 1-70/256 = 0.7265625.

Don't forget this pixels value will be in the range [0,1], so we will need to multiply this back by 256.
